About AES

The download package of AES consists of several files:

    ED.exe AES.exe HSED.exe InitWritter.exe init.pfap hs_init.pfap MSVCR100.dll state.txt versions.txt

ED.exe is the standard AES encoding program. Run this, after setting up init.pfap (see below)

AES.exe is the high-security version of ED.exe. It might take a bit longer to run as it essntially encodes the file twice: the first time with random passwords (that are then appended) to the file, and then with the passwords you specified. It is far stronger and is on par with back and military encryptions, but is more resource-intensive.
